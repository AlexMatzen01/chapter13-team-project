# Colony Command Control  
### Chapter 13 Team Programming Challenge

A C++ colony management system designed to simulate and maintain a remote off-world settlement using modular design, object-oriented programming, and dynamic data handling.

---

## Project Overview  
This software manages a space colony by tracking colonists, resources, structures, and daily operations. It reads and writes persistent data files, simulates daily events, and provides a menu-driven interface for full control of the colony.

## Features  

- Menu-driven interface with submenus  
- Daily simulation system  
- Resource consumption and depletion  
- Colonist health management  
- Structure durability and repair system  
- Random events 
- Sorting (health, resources, etc.)  
- Persistent save/load system
